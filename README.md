# 📂 Projects Included

## 🔹 Task 1: Jenkins Remoting Project

### 📖 Overview

This project demonstrates the setup and configuration of **Jenkins Remoting**, enabling a Jenkins Master to communicate securely with remote agent nodes. The implementation allows build jobs to be distributed across multiple machines, improving scalability and resource utilization.

### 🎯 Objectives

- Set up Jenkins Remoting to connect remote Jenkins nodes.
- Configure secure communication between Master and Agent.
- Distribute build workloads across remote machines.
- Execute jobs on different architectures remotely.
- Improve security using node isolation.
- Gain hands-on experience with Jenkins remote execution capabilities.

### 🛠 Technologies Used

- Jenkins
- Java
- Windows Command Prompt
- Jenkins Agent (Remoting)
- Git
- GitHub

### ⚙ Features

- Jenkins Master-Agent Architecture
- Remote Node Configuration

- Job Scheduling on Remote Nodes
- Node Isolation

# 🐳 Task 2: Web Server using Docker

### 📖 Overview

This project demonstrates the deployment and management of a web server using Docker containers. Docker was used to containerize the Nginx web server, providing a lightweight and isolated environment for web application deployment.

### 🎯 Objectives

- Learn Docker containerization basics.
- Deploy a web server inside Docker containers.
- Understand Docker container lifecycle.
- Monitor container health.
- Troubleshoot Docker containers.
- Explore container-based application deployment.

### 🛠 Technologies Used

- Docker Desktop
- Docker Engine
- Nginx
- Windows 11
- Command Prompt
- Docker Hub

### ⚙ Features

- Docker Installation & Configuration
- Docker Hub Integration
- Nginx Container Deployment
- Container Lifecycle Management
- Resource Monitoring
- Log Inspection
- Container Health Monitoring

### 📚 Docker Commands Used

```bash
docker --version

docker run hello-world

docker login

docker pull nginx

docker images

docker ps

docker rm -f mywebserver
```
